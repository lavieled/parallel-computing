# parallel-computing

ECE 046279 — Accelerated Parallel Computing (Technion, Spring 2026).

Source-only course homework: C++ concurrency, lock-free 1P1C queues, and smart-pointer connection pooling.

## Environment

- WSL2 + Ubuntu (recommended)
- `g++` (C++11 for hw0–hw1 ex1–3; C++14 for hw1 ex4)
- Optional: Intel TBB (`libtbb-dev`) for later course work
